
# IMS-GUI

This is a gui-based version of my java inventory management System

*Currently only works for Mac OS file Systems*

## Demo


https://github.com/user-attachments/assets/540df96b-7653-4f65-b916-92b2208fb922


## Current Features

- Add/Remove Products
- Updating Quantity by adding with the same Product ID
- Searching For products
- View Table for all products
- Persistent storage between uses via Gson
- Remove product bvy product ID
- Mac OS Notification Daemon Integration
